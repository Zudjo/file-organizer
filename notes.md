
# FUNZIONI

Ciclando recursivamente una cartella, i file possono essere:

- organizzati in cartelle, per
  -- estensione
  -- data di creazione
  -- data di modifica
- rinominati, per data di creazione
- eliminati, per
  -- copie esatte
  -- directory vuote

# COMPONENTS

- UserTalker
  -- prints output
  -- gets input
- FileTalker
  -- executes functions on files
- Main
  -- cycles through directory executing chosen function
